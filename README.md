# 量化交易系统

基于技术指标的自动化趋势交易分析工具。

## 设计理念

本系统基于"趋势投资"理念：
- 投资没有永恒定理，价值面不总是能成为所谓的底线
- 从消息到资本行为再到市场反应，都可看作是存在连续段的行为趋势
- 只需确保自身操作仅滞后在主力操作的一定范围内，即可获得很高的获利概率

## 技术指标

系统使用以下技术指标进行趋势分析：

1. **移动平均线 (MA)**
   - 短期、中期、长期均线
   - 判断趋势方向：均线多头排列/空头排列
   - 金叉/死叉信号

2. **RSI 相对强弱指标**
   - 超卖信号 (< 30)：可能反弹
   - 超买信号 (> 70)：注意风险

3. **MACD 指数平滑异同移动平均线**
   - 金叉：买入信号
   - 死叉：卖出信号
   - 柱状图：判断动能方向

4. **布林带 (Bollinger Bands)**
   - 触及下轨：可能反弹
   - 触及上轨：注意风险

5. **成交量分析**
   - 成交量放大：趋势确认
   - 成交量萎缩：趋势减弱

│   ├── technical_analyzer.py   # 技术指标分析

## 安装

### Python 依赖

```bash
pip install -r requirements.txt
```

### Node.js 依赖

```bash
cd web
npm install
```

## 运行

### 1. 启动后端

```bash
cd backend
python app.py
```

后端运行在 http://localhost:5000

### 2. 启动前端

```bash
cd web
npm run dev
```

前端运行在 http://localhost:3000

## 功能

1. **技术分析**
   - 实时获取股票/指数数据
   - 多维度技术指标计算
   - 综合评分和交易信号

2. **参数设置**
   - 均线周期参数
   - RSI 超买超卖阈值
   - 止损止盈比例
   - 策略评分阈值

3. **交易日志**
   - 记录交易历史
   - 统计胜率、盈亏
   - 交易原因说明

## API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/health` | GET | 健康检查 |
| `/api/params` | GET/POST | 参数设置 |
| `/api/analyze` | POST | 股票分析 |
| `/api/trades` | GET/POST | 交易记录 |
| `/api/symbols` | GET | 股票列表 |
